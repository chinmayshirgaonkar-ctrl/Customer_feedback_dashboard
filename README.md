## 📖 Overview
The **Customer Feedback Sentiment Dashboard** is an interactive, web-based analytics tool built with Python and Streamlit. It automates the process of analyzing customer reviews by applying Natural Language Processing (NLP) to classify sentiment (Positive, Neutral, Negative) and provides immediate, actionable visual insights.

## ✨ Key Features
* **Automated Sentiment Analysis:** Leverages `TextBlob` to instantly calculate polarity scores and categorize textual feedback.
* **Interactive Data Visualizations:** Utilizes `Plotly Express` to render dynamic, responsive bar and pie charts for sentiment distribution.
* **Raw Data Exploration:** Provides a clean, interactive dataframe to filter and inspect individual customer reviews alongside their calculated sentiment metrics.
* **Seamless Deployment:** Fully containerized dependency management, ready for instant deployment on Streamlit Community Cloud.

## 🛠️ Technologies & Libraries
* **Frontend / App Framework:** [Streamlit](https://streamlit.io/)
* **Data Processing:** [Pandas](https://pandas.pydata.org/)
* **Data Visualization:** [Plotly](https://plotly.com/python/)
* **NLP Model:** [TextBlob](https://textblob.readthedocs.io/en/dev/)
